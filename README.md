# Loan Prediction Using Machine Learning

## Overview

This project focuses on predicting loan approval status using Machine Learning techniques. The dataset was analyzed and preprocessed to identify important factors affecting loan approval decisions.

## Tools Used

* Python
* Pandas
* NumPy
* Matplotlib
* Scikit-learn
* Google Colab

## Project Highlights

* Performed data cleaning and preprocessing.
* Handled missing values and encoded categorical features.
* Built and evaluated multiple Machine Learning models.
* Compared model performance for loan approval prediction.
* Generated visualizations and insights from applicant data.

## Files Included

* Loan_Prediction_Dataset.ipynb
* loan_prediction_dataset.csv
* Project Report (PDF)
